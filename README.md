# 🚢 Titanic Survival Analysis

## 📌 Project Overview
This project performs Exploratory Data Analysis (EDA) on the Titanic dataset to understand survival patterns based on gender, passenger class, and age.

## 🛠 Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## 🔎 Key Insights
- Females had significantly higher survival rates than males.
- 1st Class passengers survived more than 2nd and 3rd class.
- Age influenced survival probability, with children having better chances.

## 📊 Project Includes
- Data cleaning
- Missing value handling
- Statistical analysis
- Visualizations with insights
